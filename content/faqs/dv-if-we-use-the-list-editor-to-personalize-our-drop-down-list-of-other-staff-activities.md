question: If we use the List Editor to personalize our drop down list of Other
  Staff Activities, will we be able to change the list later?
answer: You may edit existing activities, but it can become increasingly
  challenging over time. Editing to correct spelling of an activity is fine.

  records accordingly. In addition, you cannot delete any activity that has a

  InfoNet will allow you to delete an activity. This protects any data entered
  under the activity you are attempting to delete.

You may edit existing activities, but it can become increasingly challenging over time. Editing to correct spelling of an activity is fine. However, never edit one activity to reflect a different activity. For example, editing “Data Entry” to “Strategic Planning” will change all existing past records accordingly. In addition, you cannot delete any activity that has a record tied to it. You must find those records and edit or delete them before InfoNet will allow you to delete an activity. This protects any data entered under the activity you are attempting to delete.
