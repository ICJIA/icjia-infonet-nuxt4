question: When I do a search for ID numbers to enter a group session, why do
  some ID numbers appear multiple times with numbers after them?
answer: These are clients with multiple cases. The number after the ID indicates
  the case number for that client. Add the most recent case or the ID with the
  highest case number to the group.
